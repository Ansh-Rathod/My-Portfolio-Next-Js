# Found a bug?

Found a bug?

1. Check logs
2. Try to reproduce it
3. Find it in the code
4. Understand it
5. Try a different approach
6. Run again

Not working?

1. Try again.
2. ask for help in the community.

Still?

1. wait some days.

...

1. blame the framework/plugin/...
2. move on
